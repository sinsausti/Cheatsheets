Docker Cheat Sheet

Docker Basics

docker --version
docker version
docker info

Working with Containers

docker ps
docker ps -a
docker run <image>
docker run -it <image>
docker run -d <image>
docker run -p <host_port>:<container_port> <image>
docker stop <container_id>
docker start <container_id>
docker restart <container_id>
docker rm <container_id>
docker container prune

Working with Images

docker images
docker pull <image>
docker build -t <image_name> <path_to_dockerfile>
docker tag <image_id> <repository>:<tag>
docker push <repository>:<tag>
docker rmi <image_id>
docker image prune

Docker Networks

docker network ls
docker network create <network_name>
docker network connect <network_name> <container_id>
docker network disconnect <network_name> <container_id>
docker network rm <network_name>

Docker Volumes

docker volume ls
docker volume create <volume_name>
docker run -v <volume_name>:<path_in_container> <image>
docker volume rm <volume_name>
docker volume prune

Docker Compose

docker-compose up
docker-compose up -d
docker-compose down
docker-compose logs
docker-compose build
docker-compose run <service> <command>

Docker Container Management

docker logs <container_id>
docker logs -f <container_id>
docker inspect <container_id>
docker stats
docker exec -it <container_id> <command>

Docker Clean-Up

docker system prune
docker system prune -a

Examples

MySQL container:

docker run -d --name mysql1 --network home-network --ip 192.168.11.11 -e MYSQL_ROOT_PASSWORD=password -h mysql1 -p 5100:3306 -p 5101:22 -v /Users/username/Docker/mysql1/datadir:/var/lib/mysql -v /Users/username/Docker/mysql1/sshkey:/root/.ssh -v /Users/sinsausti/Docker/mysql1/confd:/etc/mysql/conf.d mysql

WordpressDB container:

docker run -d --name wordpressdb --network home-network --ip 192.168.11.12 -h wordpressdb -p 5200:3306 -p 5201:22 -v /Users/username/Docker/wordpressdb/datadir:/var/lib/mysql -v /Users/username/Docker/wordpressdb/sshkey:/root/.ssh -e MYSQL_ROOT_PASSWORD=password -e MYSQL_DATABASE=wordpressdb -e MYSQL_USER=worduser -e MYSQL_PASSWORD=wordpass mysql

Wordpress container:

docker run -d --name wordpress --network home-network --ip 192.168.11.13 -h wordpress -p 5300:80 -v /Users/username/Docker/wordpress/datadir:/var/www/html wordpress
